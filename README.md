# Smart-Zapret

Автоматическая система подбора и тестирования конфигураций WinWS для Windows.

Smart-Zapret анализирует сетевое окружение пользователя, тестирует различные стратегии обхода DPI, оценивает их эффективность и автоматически генерирует готовую конфигурацию для запуска WinWS.

---

## Возможности

### Автоматический Blockcheck (аналог blockcheck.sh от bol-van)

### Отчёты

После завершения подбора создаётся подробный отчёт с результатами тестирования.

### Автоматическая генерация конфигураций

На основе результатов создаются готовые BAT-файлы для запуска WinWS.
